# Privacy Policy

Last updated: 2026-02-19

This extension ("GLOBIS マイページ通知拡張") is designed to improve readability of notification items on specific GLOBIS VC pages.

## 1. Data Collection

This extension does not collect, store, or transmit personal data to external servers.

- No account data is collected.
- No browsing history is collected.
- No notification content is sent outside the browser.

## 2. Data Usage

The extension only reads and modifies page DOM on matched GLOBIS VC pages in order to:

- adjust notification preview display
- reduce duplicate notification visibility

All processing is performed locally in the user's browser.

## 3. Data Sharing

This extension does not sell, share, or transfer user data to third parties.

## 4. Permissions

The extension uses host access only for the matched GLOBIS VC URLs defined in `manifest.json`, solely to apply UI changes on those pages.

## 5. Security

No external API calls are required for core functionality.

## 6. Changes to This Policy

This policy may be updated when extension behavior changes. The latest version will be published in this repository.

## 7. Contact

For questions regarding this policy, please contact the extension publisher via the repository issue tracker:

- https://github.com/yokotty/globis-my-extension/issues
