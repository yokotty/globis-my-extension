# GLOBIS マイページ通知拡張

GLOBIS VC マイページの通知一覧で、メンション通知の本文を展開します。  
「詳細を見る」へ遷移せずに、メンションの内容をその場で読めるようにする拡張です。

**Features**
- タイトル末尾が「であなたにメンションしました」の通知のみ本文を全文表示
- 本文の先頭5行が一致する通知は2件目以降を非表示（メンション/リアクション対象）
- Lazy load で追加された通知も自動で展開
- ページの挙動を壊さないよう DOM を移動せずに変更

**対象ページ**
- `https://vc.globis.ac.jp/my/cm/*`

**インストール方法（Chrome）**
1. `chrome://extensions` を開く
2. Developer Mode を ON
3. "Load unpacked" をクリック
4. このリポジトリのルートフォルダを選択

**使い方**
- 対象ページを開くだけで、メンション通知の本文が自動的に展開されます
- スクロールで追加された通知も自動で展開されます

**更新方法**
1. ファイル変更後、`chrome://extensions` の更新ボタンを押す
2. 対象ページを再読み込み

**テスト**
1. `npm install`
2. `npm test`

**トラブルシューティング**
- 何も変わらない  
  - 拡張が有効か確認
  - 対象ページを再読み込み
- 新規に追加された通知が展開されない  
  - 拡張を更新してページを再読み込み

**Permissions**
- 追加の権限は不要
